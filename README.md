# My Counter

A small interactive counter built with HTML, CSS, Bootstrap, and vanilla JavaScript.  
The project focuses on DOM events, state management, and simple responsive layout.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Vanilla JavaScript (DOM events, keyboard support)

## Features
- Increase, decrease, and reset buttons
- Negative values highlighted with a red, shaking effect
- Keyboard support (Arrow Up/Down, R for reset)
- Centered card layout with responsive design
